# Unsupervised Learning

This application is used to demonstrate unsupervised learning and some of it's concepts.

---

## Technologies

This application requires all the following:

1. Python 
2. Pandas
3. hvplot
4. Pathlib
5. sklearn
6. KMeans
7. PCA
8. StandardScaler
